# Sound effects (SFX)

Drop short sound effect files in this folder and reference them by URL at runtime:

- `/sfx/click.mp3`
- `/sfx/success.mp3`
- `/sfx/error.mp3`

Notes:
- Keep files short (typically < 1s).
- Prefer `.mp3` for broad browser support.
- If you add third-party audio, keep the license/attribution requirements documented in your repo.
